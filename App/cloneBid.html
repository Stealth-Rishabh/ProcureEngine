The purpose of RFI/RFQ report is to see the details of RFI/RFQ responses received from vendors using our sourcing portal. The report can be taken out using one or many filtration parameters. The filter parameters are Report for (RFI/RFQ), Vendor Name, Subject, from/to dates.

Click on go button will display the result in grid having RFI/RFQ Reference No., Deadline Date, Vendor Name and Subject.

Clicking RFI/RFQ Reference No. will open a screen showing detailed information of RFI/RFQ.
