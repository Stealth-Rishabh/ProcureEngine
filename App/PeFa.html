Help

Using this page user can configure bids related to Scrap Sale area and invite vendors to participate on specified date and time.
There are three steps to complete this process.You may complete one step at a time by using “Save & Continue” feature and fill other details later by clicking activity displayed as “Saved as Draft” in pending activities. All the mandatory information is marked with * sign.

Details required at every step are given below:

Step -1: "Bid Details"–In this step user needs to fill all generic information related to product & services Bid. Individual columns with their relevance are given below

  • Event Subject: Meaningful short subject for the event.This is used as a subject for all related email notifications & reports.
  • Event Description: Detailed information about the event.
  • Event Duration(In minutes): Minutes given to vendors for participating/quoting price.
  • Event Date: Date when the Event will be started.
  • Currency: Currency in which amount should be quoted. e.g. INR (Indian rupees).
  • Current Conversion Factor: Conversion rate of currency at the time of event configuration.
  • Terms and conditions : PDF file which includes all terms and conditions for the event.
  • Attachments: You can upload supporting document related to event (This field is optional).
  • Approvers:Select name of approvers who will approve event.

Once all mandatory details are filled click on Save and Continue to move to next step.


Step-2: "Setup Parameters"–In this step user needs to fill product/services details against which vendors will be quoting prices/rates. Individual columns with their relevance are given below.

  • Short Name: Specify short name of product/services which would be visible at the time of event participation.
  • Target Price: which we expect to reach as a result of auction process.
  • Quantity: Specify number of units required.
  • UOM: unit of measurement Like number, kg, liter etc.
  • Description: Detailed information about the product/services.
  • Contract duration: Enter the time frame of the contract signed.
  • Dispatch Location: Name of the location where item/services need to be delivered.
  • Ceiling/MIN. Price: Enter the MIN price for event above which vendors will not be able to quote.
  • Mask from vendor: By checking this Start Unit Price will not be visible to vendors.
  • Increament on: You can define the increament in price would be based on Amount or percentage.

Step-3: "Invite Participants"–User can select participants whom he wants to send invite to participate in event. User can search from the list by typing participants name in search text box or he can check select all in case he wants to invite all available vendors.

When clicked on “Submit” system will send email alert having login id, password and bid specific details to selected vendors, their companies and approvers involved post event closer.
