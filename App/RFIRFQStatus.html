Help

The purpose of VQ/RFI report is to see the details of VQ/RFI responses received from vendors using our sourcing portal. The report can be taken out using one or many filtration parameters. The filter parameters are Report for (VQ/RFI), Vendor Name, Subject, from/to dates.

Click on go button will display the result in grid having VQ/RFI Reference No., Deadline Date, Vendor Name and Subject.

Clicking VQ/RFI Reference No. will open a screen showing detailed information of VQ/RFI.
